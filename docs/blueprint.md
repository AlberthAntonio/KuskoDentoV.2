# **App Name**: KuskoDento

## Core Features:

- Authentication: Secure login system with user and password authentication, restricted to two superusers (doctors/admins) capable of managing other users, changing passwords, and eliminating an account
- Dashboard: Clear dashboard interface to access patients, treatments, payments, radiographs, odontogram, informed consents, appointments, and backups sections.
- Patient Management: Complete patient registration with DNI, names, last names, age, telephone, address, and registration date. Supports single patient registration with multiple visits recorded. Patients must be searchable by full name and document number.
- Treatment Management: Doctors can create and assign prices to custom treatments and assign those treatments to specific patients
- Payment Tracking: System supports payments registered by advance, balance, or cancellation. Each payment is linked to a treatment, includes the payment date, and indicates the patient's payment status.
- Radiograph Management: Upload and associate radiographs in JPG, PNG, and PDF formats to patient records with date. Utilizes integrated preview functions like zoom and PDF/image viewers.
- Data Backup and Restore: Feature on the main screen to export the database to create backups, and import databases from backup files, which includes patient data, treatment configurations, payment records, radiographs, consent documents, odontogram data, and user accounts.

## Style Guidelines:

- Primary color: Deep teal (#008080) evokes a sense of cleanliness and professionalism.
- Background color: Very light teal (#E0F8F8), to create a calm environment.
- Accent color: Blue-green (#468499) is selected to create contrast on the interface and to make calls to action.
- Body and headline font: 'PT Sans' for clarity and legibility.
- Use modern and simple icons for easy navigation.
- A clean and structured layout with clear visual hierarchy.
- Subtle transitions and animations to enhance user experience.
